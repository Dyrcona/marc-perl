NAME

MARC::Charset::Hebrew - MARC8/UTF8 mappings for Hebrew

SYNOPSIS

 use MARC::Charset::Hebrew;
 my $cs = MARC::Charset::Hebrew->new();

DESCRIPTION

MARC::Charset::Hebrew provides a mapping between the MARC8 Hebrew character set and Unicode(UTF8). It is typically used by MARC::Charset, so you probably don't need to use this yourself.

METHODS

new()

The constructor, which will return you a MARC::Charset::Hebrew object.

name()

Returns the name of the character set.

lookup()

The workhorse method that does the lookup. Pass it an a character and you'll get back the UTF8 character.

combining()

Pass it a character and you'll get back a true value (1) if the character is a combining character, and false (undef) if it is not.

getCharSize()

Returns the number of bytes in each character of this character set.
